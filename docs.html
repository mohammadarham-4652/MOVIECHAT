What'sYourMov

Documentation

Everything worth knowing about how this thing works.

This tool was built for one specific reason. Picking something to watch has turned into its own exhausting task. Scrolling through streamer homepages filled with recommended content you already dismissed, reading reviews that tell you nothing about whether you personally will enjoy the film, eventually settling on something you feel indifferent about. This is meant to fix that.

You give it a feeling. It gives you exactly three films or series that match it. That is the entire idea.

Getting a recommendation

Type anything you want into the prompt at the bottom of the screen. It does not need to be a genre or a keyword. You can describe a mood, a visual style, a tempo, a time period, a filmmaker you admire, or a concept you want to think about for two hours. The system understands language, not just taxonomy.

Once you submit your prompt, the curator returns a short analysis of your request followed by three handpicked recommendations. Each one comes with a poster, a premise, a rating, and where relevant, the number of seasons instead of a runtime.

Click any card to expand it and see the full details in the side panel.

Your library

From the detail panel you can add any title to your watchlist or mark it as watched. Both of these lists live in your sidebar under Watchlist and Watched. None of this data touches a server. It all stays on your device inside your browser's local storage, which means it persists between sessions as long as you are on the same device and browser.

If you clear your browser data it will reset. There is no cloud sync. It is intentionally simple.

The poster system

Every recommendation comes with an attempt to pull the official theatrical poster for that title. The system tries several different image sources in order. If the primary source is blocked by your network, it reroutes the request through a proxy so the visual experience stays intact. In rare cases where no poster is found, it falls back to a placeholder image. This is atypical for mainstream titles.

Web series

The curator handles both films and television series. You do not need to specify which one you want. If your prompt lends itself to a long form recommendation, it may return a series. When it does, the runtime field in the detail panel automatically changes to display the number of seasons instead.

Mobile

On smaller screens the sidebar is hidden by default. Tap the menu icon in the top left corner to open it. Tapping any item in the sidebar or tapping outside of it will close it again.

Things that will not work as expected

The system is designed for conversational prompts, not keyword searches. Typing a single word like "action" will produce reasonable results but far less specific ones than something like "crime films that take place over a single night." The more context you give it, the more precisely it can match your taste.

The service that powers the recommendations is rate limited. Under normal use this is completely invisible. If you send a very large number of requests in a short window you may see a temporary failure. Waiting a minute and retrying resolves it every time.

Your conversation history within a session informs each recommendation. If you mentioned earlier that you dislike horror, the curator will factor that in for the rest of the session. Starting a new chat resets this context entirely.
